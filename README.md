# Z-music
> 基于原生js和HTML5的Audio制作的一个音乐播放器

# 功能

- 下一曲
- 播放暂停
- 单曲循环
- 歌曲时间精度/进度条
- 点击精度条播放到指定位置
- cd的旋转


# 技术：

项目基于原生js ES6语法
用面向对象的方式编写
css预编译选择用的sass
html5的audio一些常用api
基本的webpack搭建实现模块化编程
